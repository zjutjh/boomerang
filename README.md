# Boomerang

本项目为精弘网络开发部-失物招领项目

采取前后端分离的架构

后端技术栈为php-laravel

前端-未定 项目目录在 resources/assets下

采用三分支进行开发

- 主干 master
- 后端 server
- 前端 front
